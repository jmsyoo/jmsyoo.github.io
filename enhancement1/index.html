Narrative

This enhancement focuses on clean separation of concerns and safe state transitions. I moved business logic out of the controller, introduced explicit serializers to lock response shapes, and ensured all write paths run in a single transaction with row locks. The result is easier to test, easier to review, and harder to misuse.
